#Internet Explorer Manage
```PS
IE浏览器管理
```
##用途
1.备份浏览器收藏夹的链接
2.同步浏览器的链接
```Ps
同步：删除用户文件夹下收藏夹的所有链接
复制当前文件夹下的last下的所有链接到用户文件夹下收藏夹
```
##操作
1.备份链接.bat 为 备份浏览器收藏夹的链接

2.同步链接.bat 为 同步浏览器的链接

##说明
1.备份链接.bat
```list
last
Favorites→last
last→xx_xx_Favorites.7z
```
2.同步链接.bat
```list
Favorites
Favorites\Links
last→Favorites
```
